7 - Processing and Visualizing External Data Sources

Keywords: data integration, Hive, Report DB, dataset, data report

Cover page description: Learn how to prepare datasets for data reports and generate reports with charts.

Overview page: This tutorial guides you through creating a data integration task with manual scheduling to synchronize employee data from EnOS Hive to the Report Database and generating a data report with the synchronized data.

Tutorial Overview: Learning how to process data synchronized from external soruces and generate a data report. 
Unit 1: Creating a target Report Database table with specific columns to receive the synchronized data.
Unit 2: Synchronizing data manually from the Hive table to the Report Database table.
Unit 3: Creating datasets with the synchronized data.
Unit 4: Generating a data report with the chart template.
